# 問7「産声の残響」の生テキスト

日本の武士道精神を静かに宿し外国人が日本人の空気と都心に根付く自動機械消費文化渇きを知らずに挨拶かの如くコインを吸い込む自動販売機による大企業飲料会社　最終的な主張は原文で途切れているが、日本は隠れたアルコール文化、飲料大国であること。だがその文化は大学デビューという言葉があるように義務教育やスポーツと軍隊教育、空気を読む文化が残した社会は侘び寂び　ではなく我慢と報酬による生産と消費のシーソーパレード（バブル期のディスコ🕺をイメージ※筆者は当時未だ産まれていないためあくまでイメージ）

サントリー、アサヒ、CMや電車の中吊り広告、昭和、平成、令和と社会の常識は移ろい変化してきた中でアルコール業界だけは根強く呑みの文化が根付いている。

ドラッグカルチャーへの知識不足、偏見、に対してアルコールは優遇されすぎていないか？アルコール批判をしたいわけではない、戦後に失った日本固有の美学、文化、精神を引き出しの一つとして社会の日本人の空気の一つの香りとしてもちつもたれず淡い侘び寂びの境界は多様で雅な嗜みを見つけ合える友と四畳半のデコ部屋が淡い空気の世界を繋がり始める。深井戸の産声と同じいのちの振動わっしょい、わっしょいYes. Yes. Yes.
